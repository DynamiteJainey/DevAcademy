Learning about JavaScript Objects

Creating a JavaScript Objects

Using an Object Literal

This is the easiest way to create a JavaScript Object. Using an object literal, you both define and create an object in one statement.

const person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"};


Using the JavaScript Keyword new

You can create a new JavaScript object using new Object()

const person = new Object(); person.firstName = "John"; person.lastName = "Doe"; person.age = 50; person.eyeColor = "blue";
